## First GA Release

Indonesian accounting application built with Spring Boot 4.0, Thymeleaf, and PostgreSQL.

### Phase 1: Core Accounting
- Chart of Accounts with hierarchical structure and Indonesian PSAK templates
- Journal templates for recurring entries
- Double-entry bookkeeping with full audit trail
- Financial reports: Balance Sheet, Income Statement, Trial Balance, General Ledger
- Client and project tracking with invoicing

### Phase 2: Tax Compliance
- Indonesian tax calculations (PPN, PPh 21/23/4(2)/15)
- Coretax-ready export format
- Telegram receipt import with AI-powered OCR
- Tax calendar with deadline reminders

### Phase 3: Payroll & RBAC
- Employee management with 17 salary components
- BPJS Kesehatan & Ketenagakerjaan calculations
- PPh 21 with TER method (2024 regulation)
- Role-based access control (Admin, Owner, Accountant, Staff, Auditor, Employee)
- Employee self-service portal for payslips

### Phase 4: Fixed Assets
- Asset register with multiple depreciation methods (Straight-line, Declining balance)
- Asset categories and location tracking
- Disposal and sale workflows with gain/loss calculation
- Monthly depreciation batch processing

### Phase 5: Inventory & Production
- Product management with categories
- FIFO and Weighted Average costing methods
- Bill of Materials (BOM) for production
- Production orders with component consumption
- Stock valuation reports

### Phase 6: Security Hardening
- AES-256-GCM encryption for PII fields
- Document storage encryption
- Password complexity (12+ chars, mixed case, numbers, special)
- Account lockout (5 attempts, 30-minute lockout)
- Rate limiting on login and API endpoints
- Comprehensive security audit logging
- GDPR/UU PDP compliance (DSAR export, anonymization)

## Dependencies

- Java 25
- Spring Boot 4.0.0
- PostgreSQL 17
- Playwright (for functional tests)

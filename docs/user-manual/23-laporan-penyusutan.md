# Laporan Penyusutan

Laporan Penyusutan (Depreciation Report) digunakan untuk melaporkan penyusutan aset tetap dalam SPT Tahunan Badan. Laporan ini sesuai format Lampiran Khusus 1A.

## Mengakses Laporan

1. Buka menu **Laporan**
2. Pada bagian **Aset Tetap & Penutupan**, klik **Laporan Penyusutan**

![Laporan Penyusutan Aset](screenshots/reports-depreciation.png)

## Komponen Laporan

Laporan menampilkan data berikut untuk setiap aset:

| Kolom | Keterangan |
|-------|------------|
| Nama Aset | Nama dan kode aset |
| Kategori | Kategori aset (Komputer, Kendaraan, dll) |
| Tgl Perolehan | Tanggal pembelian aset |
| Harga Perolehan | Nilai beli aset |
| Masa Manfaat | Umur ekonomis dalam tahun |
| Metode | Garis Lurus atau Saldo Menurun |
| Penyusutan Tahun Ini | Jumlah penyusutan untuk tahun berjalan |
| Akum. Penyusutan | Total penyusutan sejak pembelian |
| Nilai Buku | Harga perolehan dikurangi akumulasi penyusutan |

## Memilih Tahun

Gunakan dropdown **Tahun** di bagian atas untuk melihat laporan tahun tertentu.

![Memilih Tahun Laporan Penyusutan](screenshots/reports-depreciation.png)

## Mencetak Laporan

Klik tombol **Cetak** untuk membuka versi cetak dalam format Lampiran Khusus 1A SPT Tahunan Badan.

## Tips

- Pastikan semua aset tetap sudah terdaftar di menu **Aset Tetap** sebelum akhir tahun

![Daftar Aset Tetap](screenshots/assets-list.png)

- Penyusutan dihitung otomatis setiap bulan oleh sistem
- Laporan ini bisa langsung digunakan sebagai lampiran SPT Tahunan

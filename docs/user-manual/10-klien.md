# Manajemen Klien

## Pengertian

Fitur manajemen klien memungkinkan Anda mengelola data klien atau pelanggan bisnis. Klien dapat dihubungkan dengan proyek untuk tracking profitabilitas.

## Melihat Daftar Klien

1. Klik menu **Klien** di sidebar
2. Daftar klien ditampilkan dalam tabel
3. Gunakan kotak pencarian untuk menemukan klien berdasarkan nama atau kode

## Informasi Klien

| Field | Keterangan |
|-------|------------|
| **Kode** | Kode unik klien (contoh: CLT-001) |
| **Nama** | Nama lengkap klien/perusahaan |
| **Kontak** | Nama contact person |
| **Email** | Alamat email |
| **Telepon** | Nomor telepon |
| **Alamat** | Alamat lengkap |
| **Catatan** | Catatan tambahan tentang klien |

## Menambah Klien Baru

1. Klik tombol **Klien Baru**
2. Isi form klien:
   - **Kode** - Kode unik untuk identifikasi
   - **Nama** - Nama klien/perusahaan
   - **Kontak** - Nama contact person
   - **Email** - Alamat email (opsional)
   - **Telepon** - Nomor telepon (opsional)
   - **Alamat** - Alamat lengkap (opsional)
   - **Catatan** - Informasi tambahan (opsional)
3. Klik **Simpan**

## Mengubah Data Klien

1. Klik nama klien di daftar untuk membuka detail
2. Klik tombol **Edit**
3. Ubah informasi yang diperlukan
4. Klik **Simpan Perubahan**

## Detail Klien

Halaman detail klien menampilkan:

### Informasi Kontak
- Semua data kontak klien
- Tanggal pendaftaran klien

### Daftar Proyek
- Tabel proyek yang terkait dengan klien
- Status masing-masing proyek
- Nilai kontrak dan budget

### Ringkasan Keuangan
- Total nilai kontrak semua proyek
- Total pendapatan dari klien
- Total biaya terkait klien
- Profitabilitas keseluruhan

## Menonaktifkan Klien

Klien yang tidak aktif dapat dinonaktifkan:

1. Buka detail klien
2. Klik tombol **Nonaktifkan**
3. Konfirmasi penonaktifan

Efek penonaktifan:
- Klien tidak muncul di dropdown pilihan
- Data historis tetap tersedia
- Proyek terkait tidak terpengaruh

## Tips Penggunaan

1. Gunakan format kode yang konsisten (contoh: CLT-001, CLT-002)
2. Isi informasi kontak selengkap mungkin
3. Gunakan catatan untuk menyimpan informasi penting tentang klien
4. Hubungkan semua proyek ke klien untuk laporan profitabilitas yang akurat

# Login & Autentikasi

## Masuk ke Aplikasi

Untuk menggunakan aplikasi, Anda harus login dengan akun yang telah didaftarkan oleh administrator.

## Langkah-langkah Login

1. Buka halaman aplikasi di browser
2. Masukkan **Username** Anda pada kolom yang tersedia
3. Masukkan **Password** Anda
4. Klik tombol **Masuk**

Jika login berhasil, Anda akan diarahkan ke halaman Dashboard.

## Lupa Password

Jika Anda lupa password, hubungi administrator sistem untuk melakukan reset password.

## Tips Keamanan

- Jangan bagikan password Anda kepada orang lain
- Gunakan password yang kuat (kombinasi huruf, angka, dan simbol)
- Logout setelah selesai menggunakan aplikasi, terutama di komputer bersama
- Jangan centang "Remember Me" di komputer publik

## Logout

Untuk keluar dari aplikasi:

1. Klik ikon pengguna di pojok kanan atas
2. Klik tombol **Logout**

> Catatan: Selalu logout setelah selesai menggunakan aplikasi untuk menjaga keamanan data.
